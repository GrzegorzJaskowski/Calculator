# Calculator
Calculator web app written in JavaScript

## Usage
+ Download
+ Enter the folder with downloaded files from terminal
+ Run command
  ```
  npm run start
  ```
+ Enter url
  ```
  http://localhost:3000/
  ```
  
## Features
Fully functional calculator, handling basic mathematical operations
